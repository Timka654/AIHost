## Color Scheme

- **Primary**: Purple gradient (#667eea → #764ba2)
- **Success**: Green (#28a745)
- **Danger**: Red (#dc3545)
- **Info**: Blue (#17a2b8)
- **Background**: White (#ffffff)
- **Cards**: Light gray (#f8f9fa)
- **Border**: Light gray (#e9ecef)
- **Text**: Dark gray (#333333)

## Interactive Elements

- **Buttons**: Hover effect with transform
- **Cards**: Shadow on hover with lift effect
- **Auto-refresh**: Rotating icon every 5 seconds
- **Loading states**: Spinner or skeleton screens
- **Error states**: Red alert boxes
- **Empty states**: Centered icons with messages
